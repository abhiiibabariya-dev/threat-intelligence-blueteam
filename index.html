Brute Force Detection Dashboard
MITRE: T1110 | Source: Authentication Logs | Severity: High
| Detect multiple failed logins followed by success index=auth sourcetype=linux_secure | rex field=_raw "(?i)(failed|accepted)\s+password\s+for\s+(?:invalid\s+user\s+)?(?<user>\S+)\s+from\s+(?<src_ip>\d+\.\d+\.\d+\.\d+)" | eval status=if(match(_raw,"(?i)failed"),"failure","success") | stats count(eval(status="failure")) as failures count(eval(status="success")) as successes values(src_ip) as source_ips by user | where failures > 5 AND successes > 0 | sort -failures | table user failures successes source_ips
PowerShell Empire / Download Cradle Detection
MITRE: T1059.001 | Source: Sysmon | Severity: Critical
| Detect PowerShell download and execute patterns index=sysmon EventCode=1 Image="*\\powershell.exe" OR Image="*\\pwsh.exe" | where match(CommandLine, "(?i)(IEX|Invoke-Expression|DownloadString|DownloadFile|WebClient|Net\.WebClient|Start-BitsTransfer|Invoke-WebRequest|wget|curl|IO\.MemoryStream|FromBase64String)") | eval risk_score=case( match(CommandLine,"(?i)EncodedCommand"), 90, match(CommandLine,"(?i)bypass"), 85, match(CommandLine,"(?i)hidden"), 80, 1=1, 70) | stats count values(CommandLine) as commands max(risk_score) as max_risk by Computer User ParentImage | sort -max_risk
DNS Tunneling / Data Exfiltration
MITRE: T1048.003 | Source: DNS Logs | Severity: Critical
| Detect anomalous DNS queries indicating tunneling index=dns sourcetype=dns | eval query_len=len(query) | eval subdomain_count=mvcount(split(query, ".")) | where query_len > 50 AND subdomain_count > 4 | rex field=query "\.(?<base_domain>[^.]+\.[^.]+)$" | stats count avg(query_len) as avg_len dc(query) as unique_queries sum(query_len) as total_bytes by src_ip base_domain | eval total_kb=round(total_bytes/1024,2) | where unique_queries > 50 AND avg_len > 40 | sort -total_kb
Lateral Movement - PsExec & Remote Services
MITRE: T1570, T1021.002 | Source: Windows Events | Severity: High
| Detect PsExec-style lateral movement index=windows (EventCode=7045 ServiceName="PSEXESVC*") OR (EventCode=4624 LogonType=3) | eval detection_type=case( EventCode=7045, "service_install", EventCode=4624, "network_logon") | stats count values(detection_type) as detections values(ServiceName) as services by ComputerName SourceNetworkAddress Account_Name | where mvcount(detections) > 1 OR match(mvindex(services,0),"PSEXESVC") | sort -count
C2 Beaconing Detection (Statistical)
MITRE: T1071.001 | Source: Proxy/Firewall | Severity: High
| Detect periodic C2 callbacks using time-series analysis index=proxy OR index=firewall action=allowed | bin _time span=60s | stats count by _time src_ip dest_ip dest_port | streamstats window=120 current=f stdev(count) as jitter avg(count) as avg_count by src_ip dest_ip | where jitter < 2 AND avg_count > 0 | stats count dc(_time) as beacon_intervals avg(jitter) as avg_jitter by src_ip dest_ip dest_port | where beacon_intervals > 30 | eval confidence=if(avg_jitter < 0.5, "HIGH", if(avg_jitter < 1.5, "MEDIUM", "LOW")) | sort -beacon_intervals
Ransomware Activity - Mass File Modification
MITRE: T1486 | Source: Sysmon | Severity: Critical
| Detect mass file encryption indicating ransomware index=sysmon (EventCode=11 OR EventCode=2) | rex field=TargetFilename "\.(?<ext>[^.\\]+)$" | eval suspicious_ext=if(match(ext,"(?i)(encrypted|locked|crypt|enc|locky|cerber|zepto|wnry|wcry)"),1,0) | bin _time span=5m | stats dc(TargetFilename) as files_modified sum(suspicious_ext) as suspicious_files values(ext) as extensions by _time Computer Image | where files_modified > 100 OR suspicious_files > 5 | sort -files_modified

Brute Force Detection
MITRE: T1110 | QRadar AQL
SELECT sourceip, username, COUNT(*) as attempt_count FROM events WHERE LOGSOURCETYPENAME(logsourceid) = 'Linux OS' AND "eventCategory" = 'Authentication' AND "eventName" ILIKE '%failed%' GROUP BY sourceip, username HAVING COUNT(*) > 10 ORDER BY attempt_count DESC LAST 24 HOURS
Lateral Movement Detection
MITRE: T1021 | QRadar AQL
SELECT sourceip, destinationip, username, COUNT(*) as connections FROM events WHERE "eventName" ILIKE '%logon%' AND "LogonType" = '3' AND sourceip != destinationip GROUP BY sourceip, destinationip, username HAVING COUNT(*) > 5 ORDER BY connections DESC LAST 24 HOURS

Quick Start

# Clone and setup git clone https://github.com/abhiiibabariya-dev/threat-intelligence-blueteam.git cd threat-intelligence-blueteam pip install -r requirements.txt # Collect IOCs from 7 threat feeds python ioc/ioc_manager.py collect # Analyze suspicious email headers python scripts/email_analyzer.py -f suspicious_headers.txt # Check IP reputation python scripts/ip_reputation.py 8.8.8.8 1.2.3.4 # Search CVE vulnerabilities + CISA KEV python scripts/cve_checker.py search "apache log4j" python scripts/cve_checker.py kev --recent 20 # Generate forensic timeline python scripts/timeline_generator.py /var/log/auth.log /var/log/syslog # Map MITRE ATT&CK coverage and find gaps python scripts/attack_mapper.py --gaps-only # Generate threat intelligence report python scripts/threat_report.py --markdown report.md # Analyze logs for threats python scripts/log_analyzer.py --input /var/log/syslog # Network baseline and scan python scripts/network_scan.py baseline python scripts/network_scan.py scan
