Analysis & Intelligence Tools

📊 IOC Manager
Aggregate and deduplicate indicators of compromise from 7+ threat feeds. Search, export, and maintain a local IOC database with automatic deduplication.
IOCs Threat Feeds Deduplication
🔍 Hash Checker
Multi-source file hash reputation checking. Computes MD5/SHA1/SHA256 and queries VirusTotal, MalwareBazaar, and local IOC database.
VirusTotal MalwareBazaar Hash Analysis
📨 Email Header Analyzer
Parse and analyze email headers to detect phishing, spoofing, and suspicious routing. Checks SPF, DKIM, DMARC authentication with risk scoring.
Phishing Spoofing SPF/DKIM/DMARC
🌐 IP Reputation Checker
Check IPs against AbuseIPDB, VirusTotal, blocklist.de, and local IOCs. Includes geolocation, reverse DNS, and comprehensive risk scoring.
AbuseIPDB Geolocation Risk Score
🐛 CVE Vulnerability Checker
Query NVD for CVEs by keyword or ID. Cross-references CISA Known Exploited Vulnerabilities catalog for active exploitation status.
CVE CISA KEV NVD API
📈 Log Analyzer
Pattern-based log analysis across syslog, auth, and web logs. 9 detection patterns with MITRE ATT&CK technique mapping and brute force summarization.
Syslog Auth Logs MITRE Mapped
🕑 Forensic Timeline
Generate chronological event timelines from multiple log sources. Auto-detects formats, classifies events, and exports to JSON/CSV for investigation.
Forensics Multi-source Timeline
🗺 Network Scanner
Network baseline creation and anomaly detection. Identifies new connections, compares against IOC database, and alerts on suspicious activity.
Baseline Anomaly Detection IOC Matching
🎯 ATT&CK Mapper
Maps all detection rules and playbooks to MITRE ATT&CK techniques. Identifies coverage gaps and generates visual coverage reports.
MITRE ATT&CK Coverage Analysis Gap Detection
📋 Threat Report Generator
Generates comprehensive threat intelligence reports combining IOC data, detection coverage, feed status, and actionable recommendations.
Reports Markdown Automated

Quick Start

# Clone and setup git clone https://github.com/abhiiibabariya-dev/threat-intelligence-blueteam.git cd threat-intelligence-blueteam pip install -r requirements.txt # Collect IOCs from threat feeds python ioc/ioc_manager.py collect # Analyze suspicious email headers python scripts/email_analyzer.py -f suspicious_headers.txt # Check IP reputation python scripts/ip_reputation.py 8.8.8.8 1.2.3.4 # Search for CVEs python scripts/cve_checker.py search "apache log4j" # Generate forensic timeline from logs python scripts/timeline_generator.py /var/log/auth.log /var/log/syslog # Map MITRE ATT&CK coverage python scripts/attack_mapper.py # Generate threat report python scripts/threat_report.py # Scan file hashes python scripts/hash_checker.py --hash <sha256_hash> # Analyze logs for threats python scripts/log_analyzer.py --input /var/log/syslog
