# SentinelOne EDR Detection Rules

## Overview
Custom detection rules for SentinelOne using Deep Visibility queries and STAR (Storyline Active Response) rules.

## Contents
- **deep-visibility-queries/** - Power Query hunting queries for threat investigation
- **star-rules/** - STAR custom rules for automated detection and response

## Deep Visibility Query Language
SentinelOne Deep Visibility uses a SQL-like query language for searching endpoint telemetry including process, file, registry, network, and DNS events.

## STAR Rules
Storyline Active Response (STAR) rules enable custom detection logic with automated response actions including alert, kill, quarantine, and network isolation.

## MITRE ATT&CK Coverage
All queries and rules are mapped to MITRE ATT&CK framework techniques.
