# Carbon Black EDR Detection Rules

## Overview
Watchlist queries and custom rules for VMware Carbon Black EDR (formerly CB Response) and Carbon Black Cloud.

## Contents
- **watchlist-queries/** - Process, binary, and network watchlist queries for threat hunting
- **custom-rules/** - Threat intelligence feed configurations

## Query Language
Carbon Black uses a Solr-based query syntax for searching process, binary, network connection, and module load events.

## MITRE ATT&CK Coverage
All queries are mapped to MITRE ATT&CK framework techniques.
